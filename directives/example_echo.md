# Directive: Echo Message

**Goal**: Demonstrate the 3-layer architecture by "echoing" a message back to the user via a Python script.

**Inputs**:
- Message to echo (String)

**Tools**:
- `execution/echo_tool.py`

**Steps**:
1.  Define the message to be echoed.
2.  Run `python execution/echo_tool.py --message "YOUR_MESSAGE_HERE"`
3.  Verify the output is printed to the console.
